Object Move Along a Path
-----
**Intro:**  

This is a demo of a object moving along a curve path. The object is random selected from box, sphere, dodecahedron, and cylinder. The path is generated by points with modifiable coordinates. ([view](https://yiic.github.io/my-three.js-projects/objectMoveAlongPath/objectMoveAlongPath.html))

**Sources/Libraries need to be included:**
- three.js
- stats.min.js
- OrbitControls.js
